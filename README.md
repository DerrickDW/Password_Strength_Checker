# Password Strength Checker (v1.0)
Release v1.0 - initial desktop build

Small personal desktop project built with Tkinter.

## Features
- Live password strength scoring
- Debounced real-time progress bar updates
- Optional crack-time estimates
- Packaged as a standalone Windows executable (PyInstaller)

## Purpose
Personal learning project focused on:
- GUI event handling
- Debounce logic
- Separating UI from logic modules
- Packaging Python apps into EXE
